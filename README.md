# blackjack-go
Simple CLI Blackjack game written in Go! 

## Contents
- How to Use
- ToDo

## How to Use

1. Download source
2. Build "go build blackjack.go" 
3. Run executable

Note - To run without compilation: "go run blackjack.go"

For more information on installing/using Go: https://golang.org/cmd/go/
